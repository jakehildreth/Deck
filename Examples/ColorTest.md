# Color Test

---

## Testing Color Support

---

### HTML Span Style

Using standard HTML span tags with style attribute:

* This text has <span style="color:red">red text</span> in it
* This text has <span style="color:blue">blue text</span> in it
* This text has <span style="color:green">green text</span> in it
* This text has <span style="color:yellow">yellow text</span> in it
* This text has <span style="color:magenta">magenta text</span> in it
* This text has <span style="color:cyan">cyan text</span> in it

---

### Simple Color Tags

Using simple color tags as shorthand:

* This text has <red>red text</red> in it
* This text has <blue>blue text</blue> in it
* This text has <green>green text</green> in it
* This text has <yellow>yellow text</yellow> in it
* This text has <magenta>magenta text</magenta> in it
* This text has <cyan>cyan text</cyan> in it

---

### Mixed Formatting

Combining colors with other formatting:

* **Bold** with <red>red text</red>
* _Italic_ with <blue>blue text</blue>
* `Code` with <green>green text</green>
* ~~Strikethrough~~ with <yellow>yellow text</yellow>

You can also mix: **<red>bold red text</red>** and *<blue>italic blue text</blue>*

---

### Spectre Color Names

Spectre Console supports many color names:

- <grey>grey</grey>
- <red1>red1</red1>
- <green1>green1</green1>
- <yellow1>yellow1</yellow1>
- <blue1>blue1</blue1>
- <magenta1>magenta1</magenta1>
- <cyan1>cyan1</cyan1>
- <orange1>orange1</orange1>
- <purple>purple</purple>
