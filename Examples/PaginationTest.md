# Pagination Styles Demo

---

### Minimal Style

Currently using: `minimal`

Shows just the slide number in top-right

Example: **2**

---

### Fraction Style

Change `paginationStyle: fraction` to see this

Shows current/total format

Example: **3/8**

---

### Text Style

Change `paginationStyle: text` to see this

Shows full descriptive text

Example: **Slide 4 of 8**

---

### Progress Style

Change `paginationStyle: progress` to see this

Shows a progress bar with filled blocks

Example: **████░░░░░░**

---

### Dots Style

Change `paginationStyle: dots` to see this

Shows dot indicators for all slides

Example: **○ ○ ○ ○ ○ ● ○ ○**

---

### More Content

* Try changing the paginationStyle
* Options: minimal, fraction, text, progress, dots
* Set pagination: false to disable

---
